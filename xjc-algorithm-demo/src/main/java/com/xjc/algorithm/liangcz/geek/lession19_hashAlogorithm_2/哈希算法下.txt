# 哈希算法的三个应用
  负载均衡、数据分片、分布式存储

# 负载均衡
负载均衡算法很多，如轮询、随机、加权轮询、哈希算法等
哈希算法是能实现会话粘滞的负载均衡算法，将上一次会话的
所有请求分发到同一个服务器上。
针对IP 哈希，可以将请求分发到一个机器上
# 数据分片
假设一个文件很大，1T，我们无法放到一个机器上处理。
我们使用哈希算法，对机器求摸，找到对应的机器，可以将数据分在不同机器，建立哈希表。
在查找的时候我们先找到机器，再在机器球找散列
对于很多的图片，我们也可以通过这样来处理。
# 分布式存储
我们一个机器无法存储太多东西，于是对数据计算哈希，将数据分发到该机器上。
那么如何避免重新加入机器造成雪崩？
# 一致性哈希
我们假设机器n，哈希值范围[0,m]，每个机器处理区间m/n个数据
