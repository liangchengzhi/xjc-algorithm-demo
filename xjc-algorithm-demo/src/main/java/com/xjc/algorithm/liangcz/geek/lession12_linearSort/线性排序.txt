# 桶排序
1、将数据均匀地分到每个桶中，当把每个桶的元素排序好，再把数据合并起来，就是排序好的数据。
2、数据个数n，桶m，当桶的个数约等于数据个数时候。最好的时间复杂度是0(n)
3、桶排序适合外部排序，比如数据存在外部的磁盘中，无法加载到内存中排序。
可以将元素分成很多个桶

# 计数排序
1、计数排序是桶排序的特殊形式。
2、计数排序适合排序数据范围不大的场景。
3、技术排序只适应于正整数
4、时间复杂度O(n+k),k是数据范围

# 基数排序
如何给11位长度手机号码，100万的数据快速排序？
基数排序条件：
1、数据每位可以排序
2、数据位之间有递进关系
3、每位数据范围不大
4、时间复杂度O(dn)，n是维度
