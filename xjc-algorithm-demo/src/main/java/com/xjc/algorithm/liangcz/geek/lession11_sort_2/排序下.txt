# 归并排序
归并排序使用了分治思想，先把数组分解成小的部分，保证小的有序，再组成新的数组。
1、归并排序是稳定算法吗？
是
2、归并排序的时间复杂度是多少？
O(nlogn)
3、归并排序的空间复杂度？
O(n)，因为jvm会回收栈空间

# 快速排序
归并排序有个重要的缺点是它不是原地排序算法，需要占用 O(n)空间复杂度的空间。
快速排序能解决这个问题
快速排序：
选择任意的店作为pivot，小于这个点的放左边，大于的放右边，
经过一轮排序，就把左右按顺序排出来。
* 递归：
递推公式：
quick_sort(p…r) = quick_sort(p…q-1) + quick_sort(q+1, r)

终止条件：
p >= r

快排时间复杂度：nlogn
空间复杂度：O(1)
是否稳定：不稳定
