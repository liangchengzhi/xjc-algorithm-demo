# 散列表和数组
散列表利用数组按下表索引的特性，随机快速访问到元素。散列表是数组演化而来。
是数组的一种扩展。
# 散列函数
散列函数是计算散列表数组的函数，可以定义为hash(key)
散列函数应该满足三个特点：
1、散列值返回是一个非负整数
2、如果key1=key2,那么hash(key1)=hash(key2)
3、如果key1!=key2,那么hash(key1)!=hash(key2)
要满足3不容易，很难避免散列冲突
# 如何避免散列冲突
开放寻址法和链表法
# 开放寻址法
如果出现散列冲突，就重新找一个空闲位置
线性探测法：如果冲突，就继续往下找
二次探测：平方
双重散列：多个hash算法
装载因子=元素个数/散列表长度，为了避免冲突，应该控制装载因子大小
# 链表法：
相同元素使用，链表存放
同一个槽位的，放在一个链表里
时间复杂度O(k),k=n/m,n是元素个数,m是槽位
